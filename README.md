# Retail Store Management System

A **production-grade Retail Store backend** built using Spring Boot, featuring **JWT security, JPA/Hibernate, Spring Batch, global exception handling, enterprise logging, monitoring**, modular architecture, and full testing coverage.

---

## 🚀 Features

### **🧩 Core Modules**
- Products  
- Categories  
- Users & Authentication  
- Orders  
- Cart  
- Inventory  
- Payments  

---

## 🏗️ Enterprise Architecture  
- Layered Design (**Controller → Facade → Service → Repository**)  
- Clearly defined **module boundaries** using Facade Pattern  
- **DTO + Mapper + Validator** architecture  
- Reusable utility & common modules  

---

## 🔐 Security  
- JWT Authentication & Authorization  
- Login attempt tracking  
- Custom authentication filter  
- Centralized **GlobalExceptionHandler**  
- Secured Actuator endpoints  

---

## 📦 Spring Batch Jobs  
- **Product Bulk Upload (CSV)**  
- **Stock Reconciliation Job**  
- File-based job parameter resolution  
- Scheduled job triggers  

---

## 📊 Logging & Monitoring  
- Centralized logging with **SLF4J**  
- Request Logging Filter  
- Spring Boot Actuator  
- Health Checks, Metrics, Batch Monitoring  

---

## 🧪 Testing  
- JUnit 5 + Mockito  
- Unit Tests  
  - Controller  
  - Service  
  - Security  
- Integration Tests  
- System Test Plan  

---

## 🛠️ Tech Stack  
- **Java 17**  
- **Spring Boot 3+**  
- Spring Security (JWT)  
- Spring Data JPA  
- Spring Batch  
- MySQL  
- Lombok  
- Actuator  
- Maven
